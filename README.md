# Touch ID sudo auth on Mac

## How to use
1. Clone the repository
	```
	git clone git@github.com:notimre/tid-sudo-auth.git
	```
2. Run the file
    ```
    sudo ./update_sudo_pam.sh
    ```
3. Enjoy!
